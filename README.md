# yotpo-previews

This is a personal project for reproducing and testing issues with Yotpo widgets. This repository is not owned by, affiliated with, or endorsed by Yotpo.

Static reproduction pages live in `docs/` and are published to GitHub Pages for easy sharing.

## GitHub Pages

This repo deploys the `docs/` folder to GitHub Pages on every push to `main`.

**One-time setup** (repo admin):

1. Open **Settings → Pages** on GitHub.
2. Under **Build and deployment**, set **Source** to **GitHub Actions**.

After the first successful workflow run, the site is available at:

https://kenjiefx.github.io/yotpo-previews/
